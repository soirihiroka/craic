Craic Documentation
===================

Craic is a vibe IDE and agentic development environment. It brings coding
agents, editing, Git workflows, terminals, containers, and previews into one
Libadwaita application.

Start here
----------

* :doc:`getting-started` explains how to install and launch Craic.
* :doc:`features` introduces the main workflows and document previews.
* :doc:`configuration` documents workspaces, remote hosts, coding agents, and
  appearance settings.
* :doc:`development` covers building the application and this wiki.

.. note::

   Craic is currently a personal development environment. Compatibility with
   every distribution, project type, and command-line tool is not guaranteed.
